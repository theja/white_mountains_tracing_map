White Mountains Tracing Map

This application takes the uploaded spreadsheet you use for keeping track of your hiking progess and generates an interactive map colored based on tracing status. You can view the map in this browser window or download it to your device. Learn more about white mountains tracing here. If you don't already have it, you can download a spreadsheet from the website to keep track of your hiking progress.

Notes

  • This works with 29th and 30th edition workbooks.
  • The map has alignments for most of the trails in 30th edition. Some trails in the older 29th edition are missing.
  • If you find any trails missing or geometries to be wrong, please let me know here.
  • If your spreadsheet has any sensitive information, I would recommend modifying it before uploading it for security reasons.
  • The spreadsheet should follow the format of the above-mentioned spreadsheet.
  • The code accounts for some common deviations from the format but it might sometimes raise an error.
  • If an error is raised for any reason, adjust the format and try again. If the error persists, let me know here.
